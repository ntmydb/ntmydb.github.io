先进行数据处理

第一，一般实际的数据很少没有缺失值的，所以要考虑把缺失值补上；

第二，决策树训练需要特征值，也就是将特征值处理出来，有些字段不仅很难处理成特征值而且即使处理后也对训练模型没有多大帮助；

第三，决策树的特征值只能是小数或者是整数，因此字符串的特征值需要进行映射处理

以上面数据为例：

1. Name这个特征很杂很散，而且是字符串，很难映射成数值，因此抛弃掉。
2. Sex这个特征值虽然是字符串，但只有两种，很好映射，因此保留并映射成0和1。
3. Age这个特征值很明显有空缺，然后根据大佬推荐用其他Age的平均值补上空缺。
4. SibSp和Parch这两个特征值只有数字，直接取出保留。
5. Ticket问题和Name一样，抛弃。
6. Fare均为小数，但是也有缺失值的问题，采用均值补上
7. Cabin数据缺失值很多，而且也是字符串型的特征值，舍弃
8. Embarked只有C,S,Q三种字符，可以映射后当做特征值，保留

Python处理数据代码段如下：

import pandas as pd
import numpy as np
import math
def get_dataset():
    path = 'F:/机器学习/titanic/train.csv'
    data = pd.read_csv(path)
    data['Sex'] = data['Sex'].map({'male':0,'female':1})
    data['Embarked'] = data['Embarked'].map({'C':0,'S':1,'Q':2})
    age = data['Age']
    Age = np.array(age)
    tot = 0
    num = 0
    for item in Age:
        if(np.isnan(item) == False):
            tot += item
            num += 1
    average = tot/num
    for i in range(0,len(Age)):
        if(np.isnan(Age[i]) == True):
            Age[i] = average
        Age[i] = math.ceil(Age[i])
    fare = data['Fare']
    Fare = np.array(fare)
    tot = 0
    num = 0
    for item in Fare:
        if(np.isnan(item) == False):
            tot += item
            num += 1
    average = tot/num
    for i in range(0,len(Fare)):
        if(np.isnan(Fare[i]) == True):
            Fare[i] = average
    em = data['Embarked']
    Em = np.array(em)
    tot = 0
    num = 0
    for item in Em:
        if(np.isnan(item) == False):
            tot += item
            num += 1
    average = tot/num
    for i in range(0,len(Em)):
        if(np.isnan(Em[i]) == True):
            Em[i] = average
        Em[i] = math.ceil(Em[i])
    x_train = []
    for i in range(0,len(Em)):
        features = [data['Pclass'][i],data['Sex'][i],Age[i],data['SibSp'][i],data['Parch'][i],Fare[i],Em[i]]
        x_train.append(features)
    y_train = np.array(data['Survived'])
    return np.array(x_train),y_train

处理完了数据，就可以开始训练模型了，这里因为是初学者，手模一个决策树对本萌新还是太困难，所以调用了sklearn的库来解决的

from sklearn import tree
from sklearn.metrics import f1_score, accuracy_score, plot_confusion_matrix

X_train,Y_train = get_dataset()
X_test,Y_test = get_test_data()
clf = tree.DecisionTreeClassifier(random_state=2021)
clf = clf.fit(X_train,Y_train)

然后写一个评价函数来看看预测正确率是多少

def eval(clf,X_train, X_test, y_train, y_test):
    predicted = clf.predict(X_train)  # 模型预测
    accuracy = accuracy_score(y_train, predicted)
    print("训练集准确率", accuracy)
    f1 = f1_score(y_train, predicted,average='macro')
    print("训练集f1_score", f1)

    predicted = clf.predict(X_test)  
    accuracy = accuracy_score(y_test, predicted)
    print("测试集准确率", accuracy)
    f1 = f1_score(y_test, predicted,average='macro')
    print("测试集f1_score", f1)

    plot_confusion_matrix(clf, X_test, y_test,cmap="GnBu")

执行函数

eval(clf,X_train, X_test, Y_train, Y_test)

可以看到准确率不是特别好，但以这种简单粗暴的方式构建的模型来说算是不错了，有78.9%的正确率，接近80%。在外网上看到大佬已经可以达到100%的预测准确率了，还没有过拟合，太强了orz。
