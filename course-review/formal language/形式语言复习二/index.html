# 形式定义

  • 文法
    • G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)
      • V—— 为变量的非空有穷集。∀A∈V,A\forall A\in V,A∀A∈V,A 叫做一个语法变量，简称为变量，也可以叫做非终极符号
      • T—— 为终极符的非空有穷集。∀a∈T,a\forall a\in T,a∀a∈T,a 叫做终极符。V∩T=ϕV\cap T=\phiV∩T=ϕ
      • P—— 为产生式的非空有穷集合。P 中的元素均具有形式α→β\alpha\rightarrow\betaα→β，被称为产生式，读作：α\alphaα 定义β\betaβ。其中α∈(V∪T)+\alpha\in(V\cup T)^+α∈(V∪T)+，且α\alphaα 中至少有VVV 中一个元素的出现，β∈(V∪T)∗\beta\in(V\cup T)^*β∈(V∪T)∗
      • S——S∈VS\in VS∈V，为文法GGG 的开始符号

对一组有相同左部的产生式

α→β1,α→β2,...,α→βn\alpha\rightarrow\beta_1,\alpha\rightarrow\beta_2,...,\alpha\rightarrow\beta_nα→β1​,α→β2​,...,α→βn​

可以简单记为：

α→β1∣β2∣...∣βn\alpha\rightarrow\beta_1\vert\beta_2\vert...\vert\beta_nα→β1​∣β2​∣...∣βn​

读作：α\alphaα 定义为β1,\beta_1,β1​, 或者β2,...\beta_2,...β2​,...，或者βn\beta_nβn​。并且称它们为α\alphaα 产生式。β1,β2,...,βn\beta_1,\beta_2,...,\beta_nβ1​,β2​,...,βn​ 称为候选式

  • 推导：
    • 设G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S) 是一个文法，如果α→β∈P,γ,δ∈(V∪T)∗\alpha\rightarrow\beta\in P,\gamma,\delta\in(V\cup T)^*α→β∈P,γ,δ∈(V∪T)∗，则称γαδ\gamma\alpha\deltaγαδ 在GGG 中直接推导出γβδ\gamma\beta\deltaγβδ。
    • γαδ⇒Gγβδ\gamma\alpha\delta\Rightarrow_G\gamma\beta\deltaγαδ⇒G​γβδ
    • 读作：γαδ\gamma\alpha\deltaγαδ 在文法GGG 中直接推导出γβδ\gamma\beta\deltaγβδ。
    • 直接推导可以简称推导，也称推导为派生
  • 归约：
    • γαδ⇒Gγβδ\gamma\alpha\delta\Rightarrow_G\gamma\beta\deltaγαδ⇒G​γβδ
    • 称γβδ\gamma\beta\deltaγβδ 在文法GGG 中直接规约称γαδ\gamma\alpha\deltaγαδ。在不特别强调归约的直接性时，“直接归约” 可以简称为归约
  • ⇒G,⇒G+,⇒G∗\Rightarrow_G,\Rightarrow_G^+,\Rightarrow_G^*⇒G​,⇒G+​,⇒G∗​ 当成(V∪T)∗(V\cup T)^*(V∪T)∗ 上的二元关系
    • α⇒Gnβ\alpha\Rightarrow_G^n\betaα⇒Gn​β: 表示α\alphaα 在GGG 中经过 n 步推导出β\betaβ；β\betaβ 在GGG 中经过 n 步归约成α\alphaα。
    • 当 n=0 时，有α=β\alpha=\betaα=β。即α⇒G0α\alpha\Rightarrow_G^0\alphaα⇒G0​α。
    • α⇒G+β\alpha\Rightarrow_G^+\betaα⇒G+​β：表示α\alphaα 在GGG 中经过至少一步推导出β\betaβ；β\betaβ 在GGG 中经过至少 1 步归约成α\alphaα。
    • α⇒G∗β\alpha\Rightarrow_G^*\betaα⇒G∗​β：表示α\alphaα 在GGG 中经过若干步推导出β\betaβ；β\betaβ 在GGG 中经过若干步归约成α\alphaα
    • 分别用⇒,⇒+,⇒∗,⇒n\Rightarrow,\Rightarrow^+,\Rightarrow^*,\Rightarrow^n⇒,⇒+,⇒∗,⇒n 代替⇒G,⇒G+,⇒G∗,⇒Gn\Rightarrow_G,\Rightarrow_G^+,\Rightarrow_G^*,\Rightarrow_G^n⇒G​,⇒G+​,⇒G∗​,⇒Gn​
  • 语言
    • L(G)={w∣w∈T∗,S⇒∗w}L(G)={\{}w\vert w\in T^*,S\Rightarrow^* w{\}}L(G)={w∣w∈T∗,S⇒∗w}
  • 句子
    • ∀x∈L(G),w\forall x\in L(G),w∀x∈L(G),w 称为GGG 的一个句子
    • 句子www 是从SSS 开始，在GGG 中可以推导出来的终极符号行，它不含语法变量
  • 句型
    • G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)，对于∀α∈(V∪T)∗\forall \alpha\in(V\cup T)^*∀α∈(V∪T)∗，如果S⇒∗αS\Rightarrow^*\alphaS⇒∗α，则称α\alphaα 是GGG 产生的一个句型
    • 句型α\alphaα 是从SSS 开始，在GGG 中可以推导出来的符号行，它可能含语法变量

# 文法的构造

  • 等价
    • 设有两个文法G1G_1G1​ 和G2G_2G2​，如果L(G1)=L(G2)L(G_1)=L(G_2)L(G1​)=L(G2​)，则称G1G_1G1​ 和G2G_2G2​ 等价
  • 文法G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S) 约定
    • 一个文法的所有产生式中包含的符号，就是文法中所有可能有用的终极符号和非终极符号
    • 所列的第一个产生式的左部就是该文法的开始符号

# 文法的乔姆斯基体系

  • 文法G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)

    • 短语结构文法（PSG）：

      • GGG 叫做 0 型文法，也叫做短语结构文法

      • L(G)L(G)L(G) 叫做 0 型语言。也可以叫做短语结构语言（PSL）、递归可枚举集

    • 上下文相关文法（CSG）：

      • GGG 是 0 型文法

      • 如果对于∀α→β∈P\forall \alpha\rightarrow\beta\in P∀α→β∈P，均有∣β∣≥∣α∣\vert\beta\vert\ge\vert\alpha\vert∣β∣≥∣α∣ 成立，则称GGG 为 1 型文法，或上下文相关文法

      • L(G)L(G)L(G) 叫做 1 型语言或上下文相关语言（CSL）

    • 上下文无关文法（CFG）：

      • GGG 是 1 型文法
      • 如果对于∀α→β∈P\forall\alpha\rightarrow\beta\in P∀α→β∈P，均有∣β∣≥∣α∣\vert\beta\vert\ge\vert\alpha\vert∣β∣≥∣α∣，并且α∈V\alpha\in Vα∈V 成立，则称GGG 为 2 型文法，或上下文无关文法
      • L(G)L(G)L(G) 称为 2 型语言或上下文无关语言（CFL）

    • 正则文法（RG）：

      • GGG 是 2 型文法
      • 如果对于∀α→β∈P,α→β\forall\alpha\rightarrow\beta\in P,\alpha\rightarrow\beta∀α→β∈P,α→β 均具有形式A→w,A→wBA\rightarrow w,A\rightarrow wBA→w,A→wB，其中A,B∈V,w∈T+A,B\in V,w\in T^+A,B∈V,w∈T+，则称GGG 为 3 型文法，也可称正则文法或正规文法
      • L(G)L(G)L(G) 叫做 3 型语言，也可称为正则语言或者正规语言（RL）

  • 文法分类

    • GGG 是短语结构文法
    • 如果所有产生式都有右边部分长度大于左边部分，那么GGG 是上下文有关文法
    • 如果所有产生式的左边部分都是单个非终极符号，那么GGG 是上下文无关文法
    • 如果所有产生式的右边部分都是以终极符号开始，且含有至多一个非终极符号（如果有非终极符号则出现在最右边），那么GGG 是正则文法
    • 如果一个文法GGG 是RGRGRG，则它也是CFG,CSGCFG,CSGCFG,CSG 和短语结构文法，反之不一定成立
    • 如果一个文法GGG 是CFGCFGCFG，则它也是CSGCSGCSG 和短语结构文法，反之不一定成立
    • 如果一个文法GGG 是CSGCSGCSG，则它也是短语结构文法，反之不一定成立
    • RLRLRL 也是CFL,CSLCFL,CSLCFL,CSL 和短语结构语言。反之不一定成立
    • CFLCFLCFL 也是CSLCSLCSL 和短语结构语言。反之不一定成立
    • CSLCSLCSL 是短语结构语言。反之不一定成立
    • 当文法GGG 是CFGCFGCFG 时，L(G)L(G)L(G) 却可以是RLRLRL
    • 当文法GGG 是CSGCSGCSG 时，L(G)L(G)L(G) 可以是RL,CFLRL,CFLRL,CFL
    • 当文法GGG 是短语结构文法时，L(G)L(G)L(G) 可以是RL,CFL,CSLRL,CFL,CSLRL,CFL,CSL

  • 定理：LLL 是RLRLRL 的充要条件是存在一个文法，该文法产生语言LLL，并且它的产生形式要么是形如：A→aA\rightarrow aA→a 的产生式，要么是形如A→aBA\rightarrow aBA→aB 的产生式。其中A,BA,BA,B 为语法变量，aaa 为终极符号

    • 证明：

      • 充分性：设有G′,L(G′)=LG^\prime,L(G^\prime)=LG′,L(G′)=L，且G′G^\primeG′ 的产生式的形式满足定理要求。这种文法就是RGRGRG。所以，G′G^\primeG′ 产生的语言就是RLRLRL, 故LLL 就是RLRLRL

      • 必要性：

        • 构造：用产生式组：A→a1A1,A1→a2A2,...,An−1→anA\rightarrow a_1A_1,A_1\rightarrow a_2A_2,...,A_{n-1}\rightarrow a_nA→a1​A1​,A1​→a2​A2​,...,An−1​→an​ 代替产生式A→a1a2...anA\rightarrow a_1a_2...a_nA→a1​a2​...an​

        • 用产生式组A→a1A1,A1→a2A2,...,An−1→anBA\rightarrow a_1A_1,A_1\rightarrow a_2A_2,...,A_{n-1}\rightarrow a_nBA→a1​A1​,A1​→a2​A2​,...,An−1​→an​B 代替产生式A→a1a2...anBA\rightarrow a_1a_2...a_nBA→a1​a2​...an​B

        • 证明L(G′)=L(G)L(G^\prime)=L(G)L(G′)=L(G)

          对于∀A∈V,A⇒G+x⇔A⇒G′+x\forall A\in V,A\Rightarrow_G^+ x\Leftrightarrow A\Rightarrow_{G^\prime}^+x∀A∈V,A⇒G+​x⇔A⇒G′+​x，因为S∈VS\in VS∈V，所以结论自然对SSS 成立

  • 线性文法：

    • 设G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)，如果对于∀α→β∈P,α→β\forall\alpha\rightarrow\beta\in P,\alpha\rightarrow\beta∀α→β∈P,α→β 均具有如下形式：A→w,A→wBxA\rightarrow w,A\rightarrow wBxA→w,A→wBx，其中A,B∈V,w,x∈T∗A,B\in V,w,x\in T^*A,B∈V,w,x∈T∗，则称GGG 为线性文法
    • 线性语言
      • L(G)L(G)L(G) 叫做线性语言

  • 右线性文法：

    • 设G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)，如果对于∀α→β∈P,α→β\forall\alpha\rightarrow\beta\in P,\alpha\rightarrow\beta∀α→β∈P,α→β 均具有如下形式：A→w,A→wBA\rightarrow w,A\rightarrow wBA→w,A→wB，其中A,B∈V,w,x∈T∗A,B\in V,w,x\in T^*A,B∈V,w,x∈T∗，则称GGG 为线性文法
    • 右线性语言
      • L(G)L(G)L(G) 叫做右线性语言

  • 左线性文法

    • 设G=(V,T,P,S)G=(V,T,P,S)G=(V,T,P,S)，如果对于∀α→β∈P,α→β\forall\alpha\rightarrow\beta\in P,\alpha\rightarrow\beta∀α→β∈P,α→β 均具有如下形式：A→w,A→BwA\rightarrow w,A\rightarrow BwA→w,A→Bw，其中A,B∈V,w,x∈T∗A,B\in V,w,x\in T^*A,B∈V,w,x∈T∗，则称GGG 为线性文法
    • 左线性语言
      • L(G)L(G)L(G) 叫做左线性语言

  • 定理：LLL 是一个左线性语言的充要条件是存在文法GGG，GGG 中的产生式要么是形如：A→aA\rightarrow aA→a 的产生式，要么是形如A→BaA\rightarrow BaA→Ba 的产生式，使得L(G)=LL(G)=LL(G)=L。其中A,BA,BA,B 为语法变量，aaa 为终级符号

  • 定理：左线性文法和右线性文法等价

    • 对任意右线性文法GGG，我们能构造出对应的左线性文法G′G^\primeG′，使得L(G′)=L(G)L(G^\prime)=L(G)L(G′)=L(G)；
    • 对任意左线性文法GGG，我们能构造出对应的右线性文法G′G^\primeG′，使得L(G′)=L(G)L(G^\prime)=L(G)L(G′)=L(G)

  • 定理：左线性文法的产生式与右线性文法的产生式混合所得到的文法不是RGRGRG

    • 证明：设有文法G15:S→0A,A→S1∣1G_{15}:S\rightarrow0A,A\rightarrow S1\vert1G15​:S→0A,A→S1∣1
    • 不难看出，L(G15)={0n1n∣n≥1}L(G_{15})={\{}0^n1^n\vert n\ge1{\}}L(G15​)={0n1n∣n≥1}。我们构造不出RGGRG\ GRG G，使得L(G)=L(G15)={0n1n∣n≥1}L(G)=L(G_{15})={\{}0^n1^n\vert n\ge1{\}}L(G)=L(G15​)={0n1n∣n≥1}
    • 因为L(G15)L(G_{15})L(G15​) 不是RLRLRL。所以G15G_{15}G15​ 不是RGRGRG。
