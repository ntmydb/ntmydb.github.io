# RL 的封闭性

  • 封闭性

    如果任意的、属于同一语言类的语言在某一特定运算下所得的结果仍然是该类语言，则称该语言类对此运算是封闭的

  • 有效封闭性

    给定一个语言的若干个语言的描述，如果存在一个算法，它可以构造出这些语言在给定运算下所获得的运算结果的相应形式的语言描述，则称语言类对相应的运算是有效封闭的
