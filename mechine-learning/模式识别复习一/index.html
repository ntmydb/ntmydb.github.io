# 一、距离分类器的一般形式

将待识别的样本xxx 分类到与其最相似的类别中

  • 输入：需要识别的样本\boldsymbol

  • 计算：x\boldsymbol{x}x 与所有类别的相似度S(x,wi),i=1,...,cS(\boldsymbol{x},w_i),i=1,...,cS(x,wi​),i=1,...,c

  • 输出：相似度最大的类别wjw_jwj​

    j=arg⁡max⁡1≤i≤cS(x,wj)j=\arg\max_{1 \le i \le c} S(x,w_j)j=arg1≤i≤cmax​S(x,wj​)

  • 关键：如何度量样本xxx 与类别wiw_iwi​ 的相似程度

# 二、最近邻分类器

每一个类别有多个训练样本：Di=x1(i),...,xni(i)D_i={\boldsymbol{x}^{(i)}_1,...,\boldsymbol{x}^{(i)}_{n_i}}Di​=x1(i)​,...,xni​(i)​

样本x\boldsymbol{x}x 与类别wi\boldsymbol{w}_iwi​ 之间的相似程度

S(x,wi)=−min⁡y∈Did(x,y)S(\boldsymbol{x},w_i)=-\min_{y\in D_i}d(\boldsymbol{x},\boldsymbol{y})S(x,wi​)=−y∈Di​min​d(x,y)

整体算法：

  • 输入：需要识别的样本xi\boldsymbol{x}_ixi​，训练样本集

    D=x1,...,xn;D={\boldsymbol{x}_1,...,\boldsymbol{x}_n};D=x1​,...,xn​;

  • 寻找DDD 中与x\boldsymbol{x}x 距离最近的样本：

优缺点：

  • 训练样本数量较多时效果良好
  • 计算量大，每次识别需要同所有训练样本计算距离
  • 占用储存空间大，需要保存所有训练样本
  • 易受样本噪声影响，最近邻算法只依赖最近训练样本，当训练样本某些特征有偏差、或标注错误时，会导致错误

# 三、K - 近邻分类器

判别方法：

如果j=arg⁡max⁡1≤i≤cki，则判别:x∈wj如果j=\arg\max_{1\le i\le c}k_i，则判别:x\in w_j如果j=arg1≤i≤cmax​ki​，则判别:x∈wj​

K 的选择：

  • 过小 —— 算法的性能接近于最近邻分类器
  • 过大 —— 距离较远的样本也会对分类结果产生作用，这样也会引起分类误差
  • 适合的 K 值需要根据具体问题来确定

优缺点：

优点:

  • 易于理解和实现
  • 训练时间短，有用的测试工具
  • 容易处理多分类
  • 非参数化

缺点:

  • 测试阶段的计算成本高
  • 易受数据分布的影响
  • 高维数据会降低 KNN 的精度（最近邻和最远邻之间的差异很小）

改进:

  • 减少测试时间（K−DK-DK−D 树）
  • 加权投票（weightedweightedweighted votingvotingvoting）
  • 改变距离度量标准
  • 重新缩放数据（归一化）
  • 降维（DimensionalityDimensionalityDimensionality reductionreductionreduction）

# 四、距离和相似度度量

距离度量：对于任意一个定义在两个矢量x\boldsymbol{x}x 和y\boldsymbol{y}y 上的函数d(x,y)d(x,y)d(x,y)，只要满足以下 4 个性质就可以成为距离度量

  • 非负性：d(x,y)≥0d(x,y) \ge 0d(x,y)≥0
  • 对称性：d(x,y)=d(y,x)d(x,y)=d(y,x)d(x,y)=d(y,x)
  • 自反性：d(x,y)=0d(x,y)=0d(x,y)=0，当且仅当x=yx=yx=y
  • 三角不等式：d(x,y)≤d(x,z)+d(z,y)d(x,y)\le d(x,z)+d(z,y)d(x,y)≤d(x,z)+d(z,y)

# 五、分类器性能评价

评价指标：

  • 拒识：只对有把握的样本判别类别，对没有把握的样本拒绝识别

  • 对MMM 个样本分类，mrm_rmr​ 个被拒识，mem_eme​ 个被分类错误

    错误率：Pe≈mem−mrP_e\approx\dfrac{m_e}{m-m_r}Pe​≈m−mr​me​​，拒识率：P_r\approx\dfrac{m_r}

  • 准确率：Acc=1−PeAcc=1-P_eAcc=1−Pe​

  • 召回率和查准率

    检索到未检索到
    相关a(TP)b(FN)
    不相关c(TP)d(TN)

  • 召回率：相关信息中被检索出来的比例Recall=\dfrac{TP}

  • 查准率：检索到的信息中，与主题相关的比例Precision=\dfrac{TP}

  • FScoreFScoreFScore—— 调和平均

    • 兼顾召回率和查准率，使用二者的调和平均

      F1=21R+1P=2RPR+P=TPTP+FP+FN2F_1=\dfrac{2}{\dfrac{1}{R}+\dfrac{1}{P}}=\dfrac{2RP}{R+P}=\dfrac{TP}{TP+\dfrac{FP+FN}{2}}F1​=R1​+P1​2​=R+P2RP​=TP+2FP+FN​TP​

评价方法：

  • 留出法：随机将训练样本集划分为不相交的两个子集

    • 两个子集分别用于训练、测试。重复 k 次取均值
    • 只能用一部分样本学习分类器，另一部分样本测试性能

  • 交叉验证：随机地分成不相交的 k 个子集

    • 每个子集的样本数量相同。使用 k-1 个训练，剩余样本测试
    • 以 k 次的平均值作为分类器的性能评价指标

  • BootstrapBootstrapBootstrap 方法

    • 从样本集 D 中有放回地抽取 n 个样本，组成一个BootstrapBootstrapBootstrap 样本集 A (A 中的样本可能重复)
    • 同样的方式得到另一个BootstrapBootstrapBootstrap 样本集 B
    • 用集合 A 训练分类器，然后测试集合 B 中的样本，得到对性能指标的一个估计
    • 上述过程重复 k 次，取 k 次的平均值作为分类器性能的评价
